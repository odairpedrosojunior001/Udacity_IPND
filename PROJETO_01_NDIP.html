A rede mundial de computadores

A web é composta, em grande parte, por documentos HTML. HTML significa HyperText Markup Language (linguagem de marcação de hipertextos) e é a linguagem que os computadores usam para interpretar páginas web. Os componentes da web incluem o cliente (seu computador e navegador), a Internet e os servidores. O cliente e os servidores interagem por meio de algo chamado Hypertext Transfer Protocol - HTTP (protocolo de transferência de hipertextos). Este link contém um exemplo destes componentes.

Elementos básicos de HTML

Os elementos de HTML são o que usamos para dizer a um navegador como o conteúdo de uma página web deve ser exibido. A maioria dos elementos HTML consiste em uma tag de abertura e uma de fechamento, com algum conteúdo entre elas.

<b>text</b> (coloca o texto em negrito)

<em>text</em> (coloca o texto em italico)

Atributos de HTML

Alguns elementos aceitam valores adicionais, chamados de atributos, nas tags de abertura. Os atributos dão informações adicionais ao navegador. Por exemplo, a tag anchor pega um atributo chamado href, que define a URL para a qual aquele texto deve direcionar o leitor.

<a href = "url">link</a>  (cria um hiperlink para o valor url do atributo href e o chama de link)

<img src="url" alt="text">  (exibe a imagem hospedada no valor "url" do atributo src, isso também pode ser o caminho para um arquivo local em seu computador).Caso a imagem não possa ser exibida, então o valor "text" do atributo alt é mostrado.

Mais elementos de HTML

<p>text</p>  (define um parágrafo. O conteúdo do elemento <p>  fica em uma ("box/caixa")

<br> (cria uma quebra de linha)

<div>content</div> (cria um elemento de recipiente de "box")

<span>content</span> (cria um elemento de recipiente "inline" alinhado)
